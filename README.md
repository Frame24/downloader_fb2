# Downloader FB2 - Загрузчик ранобэ в формате FB2

Проект для автоматической загрузки ранобэ с сайта ranobelib.me и конвертации их в формат FB2.

## Возможности

- 🚀 **Интерактивный режим** - удобный ввод параметров через консоль
- 🔧 **Неинтерактивный режим** - поддержка аргументов командной строки
- 📚 **Гибкая загрузка** - выбор начальной главы и количества глав
- 📥 **Загрузка отдельных глав** ранобэ
- 📖 **Загрузка полных книг**
- 🔄 **Автоматическая конвертация** в формат FB2
- 📁 **Продолжение загрузки** с определенной главы
- ⚡ **Обработка ошибок** и повторные попытки
- 🎯 **Настраиваемые параметры** - количество повторов, папка сохранения

## Установка

1. Убедитесь, что у вас установлен Python 3.7+
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

## Использование

### Интерактивный режим

Запустите скрипт без параметров для интерактивного ввода:

```bash
python download_new_book.py
```

Скрипт запросит:

- URL книги с ranobelib.me
- Номер начальной главы (Enter для начала с 1)
- Количество глав для загрузки (Enter для всех доступных)

### Неинтерактивный режим

#### Загрузка всех глав с начала

```bash
python download_new_book.py --url "URL_КНИГИ"
```

#### Загрузка с определенной главы

```bash
python download_new_book.py --url "URL_КНИГИ" --start НОМЕР_ГЛАВЫ
```

#### Загрузка определенного количества глав

```bash
python download_new_book.py --url "URL_КНИГИ" --start НОМЕР_ГЛАВЫ --count КОЛИЧЕСТВО
```

#### Дополнительные параметры

```bash
python download_new_book.py --url "URL_КНИГИ" --start 10 --count 5 --retries 5 --output "custom_folder"
```

**Параметры:**

- `--url` - URL книги с ranobelib.me (обязательный для неинтерактивного режима)
- `--start` - номер начальной главы (по умолчанию: 1)
- `--count` - количество глав для загрузки (по умолчанию: все доступные)
- `--retries` - количество повторных попыток при ошибке (по умолчанию: 3)
- `--output` - папка для сохранения (по умолчанию: results/SLUG)

### Примеры

```bash
# Интерактивный режим
python download_new_book.py

# Загрузить все главы с 1
python download_new_book.py --url "https://ranobelib.me/ru/book/116902--is-this-hero-for-real?section=chapters&ui=6052250"

# Загрузить главы 10-14
python download_new_book.py --url "https://ranobelib.me/ru/book/116902--is-this-hero-for-real?section=chapters&ui=6052250" --start 10 --count 5

# Загрузить с главы 100 до конца
python download_new_book.py --url "https://ranobelib.me/ru/book/116902--is-this-hero-for-real?section=chapters&ui=6052250" --start 100
```

## Структура проекта

- `ranobe2fb2/` - основной модуль для работы с ранобэ
- `manga2fb2/` - модуль для работы с мангой (в разработке)
- `download_new_book.py` - основной скрипт загрузки
- `tests/` - тесты проекта
- `results/` - папка с результатами загрузки

## Формат выходных файлов

- Отдельные главы сохраняются как `НОМЕР_ТОМ_НАЗВАНИЕ.fb2`
- Полная книга сохраняется как `SLUG_полная_книга.fb2`

## Тестирование
